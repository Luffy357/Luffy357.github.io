Python 爬虫

发表于 2019-08-25 | 分类于 Python

爬虫就是获取网页并提取和保存信息的自动化程序
简单来说爬虫分这基本

  • 获取网页
    • 通过 urllib、requests 等库。模拟浏览器向目标网站发出请求，
      得到网站返回的网页数据的源代码
  • 提取信息
    网页缘代码中的信息很多，但我们只想要拿到我们需要的，就需要对源代码进行分析提取我们需要的
    数据。可以使用 re、Beautiful Soup、qyquery、lxml 等库
  • 保存数据
    提取信息后，一般将数据保存到某处。保存数据可以是多种形式的，比如 txt、json或保存到数据库

爬虫基本思路就是 获取目标页面的源代码，解析 html 提取数据。保存数据
