Matplotlib

发表于 2019-08-29 | 分类于 Python

matplotlib 是一个 Python 2D 绘图库，简单易于使用


基本配置

“””
linestyle 线的样式
linewidth 线的宽度
color 线的颜色
alpha 线的透明度

线的样式

‘-‘ 实线
‘–’ 短线
‘-.’ 短点相间线
‘:’ 虚线

折点样式

'-' solid line style
'--' dashed line style
'-.' dash-dot line style
':' dotted line style
'.' point marker
',' pixel marker
'o' circle marker
'v' triangle_down marker
'^' triangle_up marker
'<' triangle_left marker
'>' triangle_right marker
'1' tri_down marker
'2' tri_up marker
'3' tri_left marker
'4' tri_right marker
's' square marker
'p' pentagon marker
'*' star marker
'h' hexagon1 marker
'H' hexagon2 marker
'+' plus marker
'x' x marker
'D' diamond marker
'd' thin_diamond marker
'|' vline marker
'_' hline marker

使用 xticks\yticks 分别设置 x 轴和 y 轴，第一个参数为刻度。第二参数为刻度标签

使用中文设置 x、y 轴和图像标题

使用 font_manager 指定字体，fontproperties 选择字体
