Python 列表与字典

发表于 2019-08-18 | 分类于 Python

列表中可以包含任何种类的对象：数字、字符串、甚至其他列表。列表是可变对象，可以原地修改它

任意对象的有序集合

将列表转换为字符串时需要注意

map 方法

将列表中的每个元素应用一个函数，并把结果放到一个新的列表中

列表的分片会得到一个新的列表
对列表的索引和分片赋值,就会修改列表。
需要注意的是分片赋值，其实是先将分片区域的值删除，再在该区域插入数据。而新插入的数据数量
不需要和删除的相同。

字典

字典通过键而不是索引来读取，是任意对象的无序集合、可变长、异构、任意嵌套。属于可变映射类型。

使用字典时需要注意，因为字典是无序的，所有序列运算例如分片是不可用的，键必须是不可变对象
