MySQL查询

发表于 2019-07-20 | 分类于 数据库

MySQL 的查询操作

  • 查询所有字段

  • 查询字段

  • as 给表、字段起别名
    为了更好的显示结果,可以给字段起别名。但需要注意别名只有在该条 SQL 语句中有效

  • 完整的查询语句
    在多表查询时,为避免混淆需要加上表名或数据库名

  • 查询去重 DISTINCT

  • 条件查询 WHERE

    • 比较运算符

    • 逻辑运算符
      逻辑运算符 ‘and’、’or’、’not’。与或非

  • 模糊查询 LIKE
    使用’百分号’、’下滑线’代替任意字符。百分号表示可有可无一个或者多个字符
    一个下滑线表示一个字符

  • 范围查询

    • IN 表示在一个非连续的范围内

    • BETWEEN 表示在某个范围之间

    • 空判断
      做空判断时不可以用比较运算符，因为 NULL 是一种特殊的值。不能进行比较要用 is 判断

  • 排序
    ASC: 从下到大为升序，DESC: 从大到小为降序。
    ORDER BY 默认为升序排序

  • 聚合函数
    COUNT 总数统计，以行作为单位统计个数

    MAX 最大值

    MIN 最小值

    SUM 求和

    AVG 平均值

  • 分组
    GROUP BY 分组的目的就是为了进行聚合统计

    • HAVING 分组后进一步的筛选
      分组之后不能使用 WHERE 做条件筛选只能用 HAVING

  • 分页
    LIMIT 指定跳过一些数据，然后显示多少条。

  • 连接查询
    连接查询 将两个表中的数据按照设置的连接条件进行筛选, 符合连接条件的数据才能够被筛选出来
    table1 INNER JOIN table2 ON 条件

    • 外连接查询
      使用 INNER JOIN 连接查询，不符合条件的数据将不显示。如果想显示就可以使用外连接
      LEFT JOIN 左外连接查询，不符合条件的将使用 NULL 填充

    RIGHT JOIN 右外连接，与左连接的意义相反。

  • 子查询
    指在一条select语句中，嵌入了另外一条select语句，那么被嵌入的select语句称之为子查询语句

    • 标量子查询：子查询语句返回的结果是一行一列

    • 列子查询：得到的数据是一列多行

    • 行子查询：得到一行多列
      行子查询需要构建行元素
      行元素：字段元素是指一个字段对应的值，行元素对应的就是多个字段：多个字段合起来作为一个元素参与运算，把这种情况称之为行元素

    • 表子查询：得到多行多列，充当数据源
