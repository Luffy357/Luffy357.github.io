MySQL的基本操作

发表于 2019-07-16 | 分类于 数据库

数据库(database)：保存有组织的数据的容器，数据库有表(table)组成，每个表又由一个或者多个列(column)组成。MySQL 数据库是一种关系型数据库。关系描述的是两个实体之间的对应规则，包括(一对一、一对多、多对多)。在关系型数据库中一行就是一个对象。

关系型数据库的设计都遵守‘三范式’

  • 第一范式： 列不可拆分
  • 第二范式： 唯一标识
  • 第三范式： 引用主键
    后一个范式都是在前一个范式的基础上建立的。
